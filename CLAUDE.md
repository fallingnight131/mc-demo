# Minecraft Demo Project

## Goal

Build a simple playable Minecraft-style browser game.

The goal is to create a demo, not a full Minecraft clone.

## Required Features

* First-person camera
* WASD movement
* Mouse look
* Procedural terrain
* Block placement
* Block destruction
* Basic inventory
* Basic textures
* Stable gameplay

## Tech Stack

* TypeScript
* Three.js
* Vite

## Working Style

* Work autonomously.
* Do not stop to ask questions.
* Fix errors automatically.
* Test changes when possible.
* read and update progress.md in every loop
* commit git in every loop

## Priorities

1. Make it playable.
2. Make it stable.
3. Make it look good.
4. Optimize performance.
