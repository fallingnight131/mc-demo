# MC Demo

用 TypeScript + Three.js + Vite 实现的可玩 Minecraft 风格体素沙盒演示。

## 运行

```bash
npm install
npm run dev      # 开发服务器(默认 http://localhost:5173)
```

打开浏览器,点击画面开始游戏(需要指针锁定权限)。

## 操作

| 按键 | 功能 |
| --- | --- |
| W A S D | 移动 |
| 鼠标 | 视角 |
| 左键 | 破坏方块(可按住连挖) |
| 右键 | 放置方块(可按住连放) |
| 中键 | 选取准星处方块 |
| 滚轮 / 1-9 / 0 | 切换物品栏 |
| E | 背包:20 种方块点选放入当前槽位 |
| 右键 TNT | 点燃(2.2 秒引信,可连锁) |
| 空格 | 跳跃 / 水中上浮 |
| Shift | 疾跑 |
| M | 静音切换 |
| T | 时间快进 1/8 天(看日落与星空) |
| Esc | 暂停(再次点击画面继续) |

## 特性

- 程序化无限地形:丘陵、山脉(带雪顶)、湖泊、沙滩、树木、野生南瓜,
  地下按深度分布煤/铁/金/钻石四种矿石,基于种子完全确定
- 区块化渲染:16×16×96 区块按需流式生成/卸载,面剔除网格,顶点色明暗
- 运行时 Canvas 生成的像素纹理图集与等距方块图标(零外部资源)
- AABB 体素碰撞、重力跳跃、游泳与水下氛围(色调+雾)
- 体素射线(Amanatides & Woo)实现精确的方块破坏/放置/选取
- 昼夜循环(8 分钟一天):太阳月亮沿弧线运动、日出日落地平线暖光、
  夜晚星空渐显、全场景亮度/雾色随时间变化,时间随存档保存
- 渐变天空穹顶 + 柔边方形日月 + 双层视差云(夜晚变剪影),水下自动隐藏
- 进度挖掘:按住左键、按方块硬度计时,5 阶段裂纹 + 材质碎屑粒子
- 水流模拟:挖开湖岸水会涌入、可形成瀑布(持续溅水)与水洼,水面漂移动画
- TNT:右键点燃、引信闪烁、球形爆坑、连锁引爆、击退与震屏(黑曜石抗爆)
- 20 种可放置方块 + 背包界面(E),快捷栏槽位可自定义并随存档保存
- 生物:猪/羊/鸡在草地自然生成,漫步/发呆/遇台阶小跳/会浮水,
  鸡下落扑翼缓降;可击退击杀(红闪 + 白烟 + 各自的惨叫),
  TNT 爆炸也会波及;闲逛时哼哼/咩咩/咯咯
- 掉落物:破坏方块掉出可拾取的小方块,自动磁吸 + 拾取音效与槽位计数徽章
- 程序化合成音效:挖掘/破碎/放置/脚步/落地/入水/爆炸,按材质分声(零音频资源)
- 环境声氛围:风声随海拔增强带阵风起伏,靠近瀑布有流水声,水下屏蔽风声
- 手持方块显示与挥动反馈、疾跑 FOV 拉伸
- localStorage 存档:世界改动与玩家状态自动保存,可在开始界面清档重开

改进过程记录见 `progress.md`。

## 脚本

```bash
npm run build    # 类型检查 + 产物构建
npm test         # 单元测试(地形/物理/射线)
npm run verify   # 无头 Chrome 实机游玩冒烟测试(需 dev 服务器运行中)
```

`?test` URL 参数为自动化测试钩子:跳过指针锁定并暴露 `window.__game`。
